Проектирование сетей оптических линий связи для доступа в интернет с учётом максимального кол-ва абонентов, включает в себя: Проектирование , монтаж, подключение и обслуживание абонентов internet. На этапе проектирования в первую очередь учитывается кол-во потребителей интернета. Для упрощения процесса разрабатывается Web-service с привязкой к Приложению по созданию вычисления кол-ва материалов/оборудования из БД для монтажа GPON сети с привязкой к OpeNMaps(картам), по которым абонент/провайдер/любой пользователь на сайте может определить ~стоимость подключения к сети с учётом оборудования, материалов, работ, которые производят калькуляцию и вычисление подключения в Частом/Многоквартирном секторе.

(Eng.) Designing networks of optical communication lines for Internet access, taking into account the maximum number of subscribers, includes: Design, installation, connection and maintenance of internet subscribers. At the design stage, the number of Internet consumers is primarily taken into account. To simplify the process, a Web-service is being developed with reference to the Application to create a calculation of the number of materials / equipment from the DB for mounting GPON networks linked to OpeNMaps(maps), by which subscriber/provider/any user on the site can determine ~cost of connecting to the network taking into account the equipment, materials, works that calculate and calculate the connection in the Private / Multi-apartment sector.
